## 货币及货币供给

### 货币是什么

货币的实质是一种市场交易的解决方案，这种工具属性本身是货币的**价值**，而让货币具有这种工具属性的前提条件，是维持货币价格的稳定。

书中定义货币为在产品和服务支付以及债务偿还中被普遍接受的东西。

货币具有三个主要功能：交易媒介、记账单位和价值储藏，需要说明的是货币作为价值储藏手段并非独一无二，任何资产，无论是货币、股票、债券、土地、房屋、艺术品还是珠宝，都可以用来储藏财富，但是在一个伴随常规通货膨胀的社会，货币本身几乎没有价值升值空间，甚至会因为通货膨胀而贬值，人们持有货币的原因是因为流动性（即一种资产转化为交易媒介的便利程度和速度）。

### 货币供给

理解货币供给的决定机制十分重要。谁控制货币供给，哪些因素会引起货币供给的的变动，如何提高货币供给的控制力。

货币供给主要有三位参与者：一是中央银行（监管银行体系的政府机构，以下简称央行），二是银行（从个人和机构手中吸收存款并发放贷款的金融中介机构，包括且不限于商业银行），三是储户。

这三者有三种交互关系，分别是央行与银行，银行与储户，央行和储户，但实际上只有两种，央行和储户的债务关系是被动产生的，二者没有直接交互关系。

首先通过看央行的资产负债表来了解一些概念：

| 资产                 | 负债         |
| -------------------- | ------------ |
| 证券                 | 流通中的现金 |
| 向金融机构发放的贷款 | 准备金       |

负债项：

1. 流通中现金

   是央行发行发行的货币（国内就是以毛爷爷为主的**纸张**，上面印有中国人民银行字样）

2. 准备金

   所有银行都在央行开立存款账户，准备金包括在央行的存款和银行实际持有的现金，准备金是央行的负债，银行的资产，银行可以要求央行以毛爷爷偿付来履行其义务。

   准备金可以被划分为两类，法定准备金和超额准备金。法定准备金是银行必须持有的准备金，超额准备金是银行自愿持有的额外准备金

资产项：

1. 证券

   央行通过购买证券为银行体系提供准备金，从而增加其政权资产

2. 向金融机构发放的贷款

   央行还可以通过发放贷款为银行体系提供准备金，对借入的银行而言这些贷款成为贴现贷款，央行向银行收取的贷款利率被称为贴现率

基础货币$MB$等于流通中现金$C$加上银行体系的准备金总额$R$：
$$
MB=C+R
$$


央行通过**公开市场操作**与向银行发放贴现贷款**对基础货币实现控制**。

#### 货币乘数

基础货币并不能反映市场对货币的需求，真实反映市场货币需求的是银行向储户发放的贷款，这涉及到货币乘数的概念。

货币乘数$m$可以反映基础货币$MB$的变动所引起的货币供给$M$的变动，通常我们可以简单理解货币乘数的值为法定准备金的倒数，三者的关系可以表达为：
$$
M=m*MB
$$
货币乘数$m$是基础货币转化为货币供给的倍数，更具体的m的表达式如下：
$$
m=\frac{1+c}{rr+e+c}
$$


式中$rr$ 为法定准备净率，$c$为现金比率,$e$为超额准备金比率，通过该式可以推断其他因素变动对货币供给的影响：

|        |                      |            |                |              |
| ------ | -------------------- | ---------- | -------------- | ------------ |
| 参与者 | 变量                 | 变量的变动 | 货币供给的反映 | 推理         |
| 央行   | 非借入基础货币$MB_n$ | $\uparrow$ | $\uparrow$     | 基础货币增加，货币供给自然增加 |
| 央行   | 法定准备金率$rr$     | $\uparrow$ | $\downarrow$   | 货币乘数减小 |
| 银行   | 借入准备金$BR$       | $\uparrow$ | $\uparrow$     | 基础货币增加 |
| 银行   | 超额准备金           | $\uparrow$ |  $\downarrow$    | e增大，货币乘数减小 |
| 储户   | 现金持有水平         | $\uparrow$ | $\downarrow$    | 对m表达式处理后可以发现，随着c的增大，货币乘数减小 |
